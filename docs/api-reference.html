    footbal_interface v0.1.0 API Reference

    Modules

    FootbalInterface.Formatter

    Contains the logic to convert messages into the various protocols supported. Currently supports JSON and Google's Protocol Buffer.

    FootbalInterface.Messages

    Contains all the .proto schemas for Google's protool buffer format.

    FootbalInterface.Web.Controllers.NotFound

    Runs when no URL matches.

    FootbalInterface.Web.Controllers.Ping

    Basic ping controller which will always return 200. Used periodically to check if the server is alive.

    FootbalInterface.Web.Controllers.Search

    Maps the results of the search query made to HTTP and sends the response to the client.

    FootbalInterface.Web.Plugs.MetricsExporter

    Exposes the /metrics path and responds in Prometheus metrics format.

    FootbalInterface.Web.Plugs.MetricsInstrumenter

    Adds HTTP related metrics for every request.

    FootbalInterface.Web.Router

    Plug Router for all incoming requests. Feeds the metrics modules with data and dispatches the requests to the correct controllers after.

    FootbalInterface.Workflow

    Worflows represent the logic that controllers use. They implement the steps that requests need to take to be completed and return answer ready for the controllers to decode.

    FootbalInterface.Workflow.Search

    Flow of a search request. Parses the request and queries the DB.
